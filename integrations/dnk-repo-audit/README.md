# 🛡 DNK Repo Audit — Deep Security Intelligence

> Автономний аудит коду на базі LLM для виявлення складних багів та логічних помилок.

Ця інтеграція використовує [PurCL/RepoAudit](https://github.com/PurCL/RepoAudit) для проведення глибокого аналізу репозиторіїв екосистеми DNK HUB.

## 🔱 Можливості
- **Bug Detection**: Пошук Null Pointer Dereference, Memory Leaks, Use After Free.
- **Multi-lingual**: Підтримка Python, Go, C++, Java.
- **LLM-Driven**: Використання нейро-символічного аналізу для розуміння контексту коду.

## 🚀 Як використовувати
1. **Підготовка**: Запустіть `./setup.sh` для налаштування середовища.
2. **Аудит проекту**:
   ```bash
   ./audit.sh /шлях/до/проекту
   ```

## 🧬 Роль в екосистемі
Цей інструмент є "Останньою лінією оборони" перед виходом продукту на стадію комерційної експлуатації.
