# Obsidian MCP

MCP для роботи з Obsidian vault — читання і запис нотаток.

## Варіанти

1. **obsidian-mcp** (community) — через Local REST API Obsidian plugin

2. Прямий доступ до файлової системи vault (якщо на тому ж компʼютері)
